# Comando `man`

Un comando es el nombre de programa, cada comando tiene un nombre y ejecuta una tarea especifica, por ejemplo, si escribimos el comando date en la terminal, este comando nos va a regresar la fecha y hora, algo importante es que los comandos tienen opciones, est√°s se agregan frente al nombre del comando.

Cada comando tiene un manual para el usuario al que se puede acceder con el comando man:

```bash
man ls
```
